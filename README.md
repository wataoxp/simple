# Simple

## 概要
LLライブラリを用いたベアメタル(っぽい)プログラムです。

## 説明
* I2Cを使ったLCDの表示
* Lチカ
* GPIO割り込みで手動Lチカ

## Requirement
* STM32CubeIDE
* STM32G031F6P6
* Clang

* delay.h
* gpio.c

* i2c.c
